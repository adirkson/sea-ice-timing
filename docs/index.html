Welcome to sea-ice-timing¶

This project contains code used to perform statistical postprocessing on ensemble forecasts of sea-ice retreat and advance dates. A reference will be provided following publication of the article describing this methodology.

NCGR¶

Installation¶

pip install NCGR

Simple Usage¶

from NCGR import ncgr

ncgr.ncgr_fullfield(hc_netcdf, obs_netcdf, fcst_netcdf, out_netcdf, event,
                 a, b, clim_netcdf=clim_netcdf)
